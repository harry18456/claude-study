## Why

工程師同事對 Claude 的各項功能（Claude.ai、Claude Code、API、MCP、Skill 系統）缺乏系統性的了解。建立一套完整的參考文件，作為週會介紹 Claude 的知識基礎，並供日後查閱。

## What Changes

- 新增 `GENERAL.md`：Claude 概覽、模型差異、方案比較
- 新增 `CHAT.md`：Claude.ai 介面功能、Artifacts、Projects、Memory
- 新增 `COWORK.md`：協作場景、工作流程應用
- 新增 `CODE.md`：Claude Code CLI 開發工具使用說明
- 新增 `API.md`：Claude API / SDK 串接開發指南
- 新增 `MCP.md`：MCP 協議概念與外部工具整合說明
- 新增 `SKILL.md`：Claude Code Skill 系統與自訂 workflow 說明

## Capabilities

### New Capabilities

- `general-overview`: Claude 基本概念、模型家族（Haiku/Sonnet/Opus）、方案與定價比較
- `chat-interface`: Claude.ai 網頁介面功能，包含 Artifacts、Projects、Memory 使用方式
- `cowork-collaboration`: 協作場景設計與工作流程整合應用案例
- `claude-code-cli`: Claude Code CLI 工具安裝、設定、核心功能與開發工作流
- `claude-api-sdk`: Claude API 串接、Anthropic SDK 使用、模型選擇與參數設定
- `mcp-integration`: Model Context Protocol 概念、MCP Server 建立與外部工具整合
- `skill-system`: Claude Code Skill 系統架構、自訂 skill 建立與可重用 workflow 設計

### Modified Capabilities

（無）

## Impact

- 受影響的檔案：`GENERAL.md`、`CHAT.md`、`COWORK.md`、`CODE.md`、`API.md`、`MCP.md`、`SKILL.md`（全為新建）
- 無現有程式碼受影響
- 無外部 API 或依賴變更
